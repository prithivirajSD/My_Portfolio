Clinical Decision Support System

Summary: Designed a deep learning-based CDSS to predict diseases from 132 clinical symptoms using Bidirectional LSTMs.

  • Compared models (SVM, Decision Tree, KNN, Logistic Regression) using accuracy, precision, and F1-score
  • Bidirectional LSTM achieved 90.16% accuracy, outperforming other models
  • Optimized model performance via hyperparameter tuning
  • Implemented personalized medical suggestions with specialist recommendations
  • Developed a web application integrating the model, enabling users to input symptoms and receive potential diagnoses
  • Published research paper in IEEE Conference: "Clinical Decision Support System using Recurrent Neural Network"
  • Tech Stack: Python, Deep Learning, BiLSTM, RNN, Machine Learning
